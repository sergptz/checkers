# Шашки
## Документация
### Описание
Игра Шашки.
### Части системы:

1. GameSession - Прослойка для работы с данными о игре (у нас данные будут хранится в хранилище vuex). 
2. GameRuler - Класс с логикой игры, изменяющий её состояние, путём взаимодействия с GameSession
3. Хранилище vuex
